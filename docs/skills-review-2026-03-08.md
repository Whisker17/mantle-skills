# Skills Review Notes

## Mantle Network Primer Positioning

- `skills/mantle-network-primer/SKILL.md` is intentionally kept as a `reference/onboarding skill`.
- Its purpose is to clarify Mantle-specific assumptions that are easy to get wrong:
  - `MNT` as the gas token
  - inclusion versus L1-backed settlement/finality
  - Mantle-specific chain settings and onboarding links
  - public RPC limitations and live-verify boundaries
- It should not be evaluated as a pure execution operator like `mantle-defi-operator`, `mantle-risk-evaluator`, or `mantle-address-registry-navigator`.
- In SkillsBench terms, it is useful and intentional, but closer to a domain reference/onboarding artifact than an archetypal procedural skill.
